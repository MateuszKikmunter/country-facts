Project intents do design website with facts about countries and use following tech stack:
  1. Bootstrap 4 + Bootswatch (I like darkly theme and I used it before) for design and look and feel
  2. jQuery (Bootstrap requires it), but rest of JS code should be written in Vanilla JS and splitted into modules
  3. jQuery datatables for tabular data
  4. REST countries api as data source
  5. font awesome for icons
Project can contain following pages:
  1. welcome page
  2. quick search
  3. index with table showing all the countries with details button for each one
  4. details page where user can read more detailed info about particular country
  5. about page - something about the project + tech satck used and something about us
  6. contact page with simple contact form
